Weather App


Between October 2020 and April 2021, I decided to learn a bit about front-end development. On the one hand, I wanted to understand how design ideas are translated into code and on the other hand, I wanted to understand how I can work better with developers.

I came across SheCodes, an online workshop that aims to introduce women to the world of coding. I first completed the SheCodes Plus and SheCodes React modules.


Course objectives

During the course I learned about:

  • HTML
  • CSS
  • Bootstrap
  • JavaScript
  • APIs
  • Git and GitHub

How it worked

I completed daily exercises based on pre-recorded lectures and at the end of each week I handed in a piece of homework which was checked by a tutor.

At the end of the course I coded a project – a weather app – in VS Code. This project was hosted via Netlify and approved by SheCodes.



SheCodes Plus: I had to design and code a weather app to pass my coding course.

Homework week 1: I had to draw a wireframe of the weather app I planned to code in Invision.

Homework week 2: I had to convert my Invision wireframe into HTML and CSS.

Homework week 3 and 4: I had to add a bit of JavaScript to my design, asking a user to enter data via a pop-up.

Homework week 5: I had to connect my project to an OpenWeather API and add a geolocation button.

Homework week 6, 7 and 8: I had to deploy the app to Netlify, improve the code and add a weather forecast.
